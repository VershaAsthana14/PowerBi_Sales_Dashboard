📊 Sales Dashboard - Power BI Project

🔹 Overview
This project is a Sales Dashboard built using Power BI to analyze business performance.

🔹 Tools Used
- Power BI
- Excel
- DAX

🔹 Key Features
- KPI Cards (Sales, Profit, Quantity)
- Sales Trend Analysis
- Top 5 Products
- Regional Performance
- Profit Growth %

🔹 Insights
- Sales increased by 20% in Q2
- West region performed best
- Top product contributed 35% revenue
